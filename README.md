2. Download the latest Release
3. Extract the zip file
4. Run RouteSettingSimulator.exe

## Controls

- WASD - Move Camera
- Mouse - Look Around
- E - Select Hold/Bolt
- Mouse (Hold) - Rotate Selected Hold
- Tab - Toggle Controls
- Esc - Close Menus

## Requirements

- Windows/Mac (depending on your build)
- Graphics card supporting DirectX 11 or later
